Alineando con flex.

Página de Iguanas, proyecto que involucra HTML5 y CSS, aplicamos Flex, un modelo de diseño

Bootstrap.

Página de venta de paquetes de viaje. Para su creación utilizamos Bootstrap,

que proporciona una colección de herramientas y componentes CSS y JavaScript preestilizados,

como botones, formularios, barras de navegación, cuadros modales y más.

GIT/GITHUB.

Página de venta de paquetes de viajes, creada a partir de la clonación de un repositorio existente en GITHUB.

Se utilizó el terminal de Gitbash y de VSCODE.

Git es una herramienta esencial para el desarrollo de software colaborativo y la gestión efectiva del código fuente.
